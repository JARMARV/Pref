Pref will be a open source website framework anyone may use to plan certain kinds of events.

The idea is that when you plan an event with for example multiple lectures at the same time and the attendees
should be distributed equally ,each attendee can set their preferences where they want to go the least to most before the start of the event
and then an algorithm automatically chooses the ideal distribution of attendees in a way that maximizes the satisfaction of each attendee,while
taking into account certain restriction like an equal distribution or minimum and maximun number of attendees in one instance.

This is originally intended for use in company internal or school events but could theoretically be modified to work with events open to the public.

Note: the project is still work in progress and currently far away from a working prototype
